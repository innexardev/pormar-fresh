# Fresh Box UI/UX

## Priority surfaces

- Home: proposta "cortado no dia", entrega 2x/semana
- Cardapio: combos em destaque, peso/porcao visivel
- Checkout: selecao clara de terca vs sexta
- Pedido: timeline de status amigavel

## Visual

- Verde (`fresh-600/700`), fundo claro, cards limpos
- Mobile-first, botoes grandes no carrinho fixo

## a11y

- Labels claros em formulario de entrega
- Contraste adequado em CTAs
